# job-search-backend-api
Restful API build with Restify in Node JS


## API

* Get all jobs : [https://job-search-backend-api.herokuapp.com/api/jobs/list](https://job-search-backend-api.herokuapp.com/api/jobs/list)

* Get particular job : [https://job-search-backend-api.herokuapp.com/api/jobs/:id](https://job-search-backend-api.herokuapp.com/api/jobs/:id)

* Filter job with title : [https://job-search-backend-api.herokuapp.com/api/jobs/filter?title=Senior%20Developer](https://job-search-backend-api.herokuapp.com/api/jobs/filter?title=Senior%20Developer)
